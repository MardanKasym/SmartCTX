# SmartCTX - Умная столовая в твоем смартфоне 📱

## Обзор проекта (Mobile MVP)
SmartCTX — это проект мобильного приложения, разработанный с помощью React Native и Expo. Основная миссия — перенести процесс обслуживания столовой в смартфоны: никаких очередей, быстрая навигация, интерактивное меню и моментальная сборка заказа. Текущая версия (Minimum Viable Product) включает в себя весь фронтенд-каркас и базовое управление состоянием логики покупок.

## Главные фичи приложения
- **Приложение на обе платформы:** Использование Expo (SDK 54) позволяет запускаться сразу на iOS и Android.
- **Главный экран (Home):** Быстрый доступ к разделам, статистика.
- **Интерактивное Меню (Menu):** Сетка блюд, цены в тенге ₸, калорийность и категории с возможностью фильтрации.
- **Корзина (Cart):** Глобальное управление корзиной на уровне всего приложения. Добавление/Удаление товаров, мгновенный расчет суммы заказа. Работает через React Context API.
- **Оформление заказа (OrderStatus):** После клика "Оформить заказ" система генерирует билет с номером заказа, временем ожидания и местом выдачи.
- **Glassmorphism UI:** Использование темной темы `#0f172a` и прозрачных полуматовых панелек, создающих современный объемный вид.

## 👨‍💻 Инструкция для студентов: Как устроен код?

Вся магия происходит внутри папки **`mobile/`**. Вот как она работает:

### 1. Архитектура файлов (Где что лежит?)
* `App.js` — **Самый главный файл!** Точка входа в приложение. Здесь же находится Tab Navigator (те самые нижние 4 кнопки: Главная, Меню, Корзина, Профиль). А еще там подключен карточный провайдер `<CartProvider>`.
* `screens/` — Это ваши экраны приложения. Если хотите поменять текст на главной странице — вам сюда.
  * `Home.js`, `Menu.js`, `Cart.js`, `OrderStatus.js`, `Profile.js`. 
* `components/` — Повторяющиеся блоки кода (компоненты). 
  * Карточка каждого блюда отрисовывается через `FoodCard.js`. Модуль вашей корзины (счетчик и фото) — `CartItem.js`, а бегунок категорий меню это `CategoryFilter.js`.
* `context/` — Состояние глобальной даты.
  * Файл `CartContext.js`. Когда вы нажимаете кнопку "+" в меню, этот файл запоминает, что товар добавлен. Значок "корзины" внизу меняет цифру благодаря именно этому файлу!
* `data/` — Временная База Данных.
  * Файл `menuItems.js` (можете добавлять свои пиццы и салаты: тут все названия и цены).

### 2. Как запустить этот проект у себя на компьютере? 🚀

1. Откройте терминал (Command Prompt \ Terminal).
2. Перейдите в папку с приложением:
   ```bash
   cd mobile
   ```
3. Установите все "детали" с помощью Node.js:
   ```bash
   npm install
   ```
4. Запустите сервер:
   ```bash
   npx expo start
   ```

### 3. Как проверить свою работу на телефоне?
* **Для iPhone:** Скачайте в App Store приложение **Expo Go**. Просто откройте стандартную камеру и наведите на появившийся в компьютере QR-код.
* **Для Android:** Скачайте в Google Play **Expo Go**. Нажмите "Scan QR code" внутри самого приложения.

> **💡 Секрет от разработчиков (Fix EMFILE error):** 
> Если при запуске сервер падает с ошибкой `EMFILE: too many open files` – это значит, что у MacOS стоит лимит на слежку за файлами. Напишите в терминале: `ulimit -n 65536`.
